# Evolution Alert

Aplicação para gerar e renderizar QR codes de autenticação para instâncias WhatsApp via Evolution API.

## 🚀 Sobre

Evolution Alert é um assistente web que facilita a geração de QR codes para conectar instâncias de WhatsApp à Evolution API. Os parâmetros de conexão (domínio, instância, número, tokens) são enviados por parâmetros na URL, criptografados para segurança. A aplicação oferece uma interface simples para exibição do QR code e um assistente para construir URLs configuráveis.

## 📋 Funcionalidades

- **Geração de QR Code**: Renderiza QR code dinamicamente da Evolution API
- **Verificação de Status**: Valida se a instância já está conectada
- **Assistente de Links**: Gera URLs criptografadas para compartilhamento
- **Regeneração de QR**: Permite solicitar novo código se necessário
